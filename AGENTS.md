# AGENTS.md

## 项目目标

本仓库以现有期刊 LaTeX 模板为基础，逐步简化为适合论文初稿写作、Pandoc 转 Word 审阅、以及后续按期刊格式整理的模板。

## 工作原则

- 优先保持现有期刊模板文件可追溯，不随意删除或重写原始模板。
- 简化模板时，应优先拆分出更小、更通用的 manuscript 模板，而不是直接破坏期刊示例文件。
- 面向 Pandoc 转换的内容应尽量使用标准 LaTeX 结构，例如 `\section`、`\subsection`、`\caption`、`\label`、`\ref`、`\cite`、普通表格和外部图片。
- 避免在 Pandoc 友好模板中引入复杂自定义宏、重型 class 配置、特殊浮动体和难以转换的排版技巧。
- 新增脚本或说明文档时，应写清楚用途、输入输出和推荐命令。

## Git 约定

- commit message 使用中文。
- 提交应尽量小而清晰，一次提交只表达一个主要意图。
- 不要提交编译生成物，除非它是项目明确需要保留的参考产物。
- 修改现有期刊模板前，先确认是否应该保留原文件并另建简化版本。

## 文件约定

- 原始期刊模板文件保留其原名，作为参考基线。
- 简化模板建议放入独立目录，避免与原始期刊模板混杂。
- Pandoc 的 `reference.docx` 应与对应模板或说明文档放在同一工作目录，便于复现转换流程。
