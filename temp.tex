\begin{abstract}
本文给出一个极简LaTeX论文手稿示例，目标是在保留论文常用语义元素的同时，尽量避免复杂期刊排版命令，从而提高Pandoc转换为Word文档时的稳定性。模板包含标题、作者、摘要、关键词、章节层级、图表、公式、交叉引用和参考文献等常见内容。 This example provides a minimal LaTeX manuscript template. It keeps common semantic elements for academic writing while avoiding complex journal-specific layout commands, so that Pandoc can convert the manuscript to Word more reliably. The template includes the title, authors, abstract, keywords, section hierarchy, figures, tables, equations, cross references, and references.
\end{abstract}

\section{引言 Introduction}
\label{sec:introduction}

论文初稿通常更重视内容结构和审阅便利性，而不是期刊最终排版。为了便于转换为Word，建议手稿中优先使用标准LaTeX命令，例如章节命令、图表环境、公式环境和普通引用命令。 A draft manuscript usually focuses on content structure and review convenience rather than final journal layout. To make Word conversion easier, the manuscript should prefer standard LaTeX commands, including section commands, figure and table environments, equation environments, and ordinary citation commands.

本文的结构如下：第\ref{sec:method}节介绍基本写法，第\ref{sec:results}节给出图表和公式示例，第\ref{sec:conclusion}节总结使用建议。 The structure of this manuscript is as follows: Section \ref{sec:method} introduces the basic writing pattern, Section \ref{sec:results} gives examples of figures, tables, and equations, and Section \ref{sec:conclusion} summarizes usage suggestions.

\subsection{手稿结构 Manuscript Structure}
\label{subsec:structure}

手稿结构建议保持简单。标题、作者、摘要和关键词放在正文开头；正文使用\verb|\section|、\verb|\subsection|和\verb|\subsubsection|表示层级；图、表、公式使用标准浮动体和数学环境。 The manuscript structure should remain simple. The title, authors, abstract, and keywords are placed before the main text; the body uses \verb|\section|, \verb|\subsection|, and \verb|\subsubsection| to express hierarchy; figures, tables, and equations use standard floating and mathematical environments.

\subsubsection{三级标题示例 Third-Level Heading Example}
\label{subsubsec:third-level}

三级标题可用于细分方法、实验条件或推导步骤。若后续需要投递期刊，可在定稿阶段再迁移到期刊模板，而不是在初稿阶段混入大量版式命令。 A third-level heading can be used to divide methods, experimental conditions, or derivation steps. If the manuscript later needs to be submitted to a journal, it can be migrated to the target journal template at the final stage instead of mixing many layout commands into the draft.

\subsection{公式与引用 Equations and Citations}
\label{subsec:equation}

公式建议使用标准\verb|equation|环境。式\ref{eq:sample}给出一个简单的线性模型： Equations should use the standard \verb|equation| environment. Equation \ref{eq:sample} gives a simple linear model:

\begin{equation}
  y = ax + b
  \label{eq:sample}
\end{equation}

其中，$x$为自变量，$y$为因变量，$a$和$b$分别为模型参数。正文中可以继续引用式\ref{eq:sample}，但不建议使用期刊模板专用的自定义引用命令。 Here, $x$ is the independent variable, $y$ is the dependent variable, and $a$ and $b$ are model parameters. The body text can continue to cite Equation \ref{eq:sample}, but journal-template-specific custom reference commands are not recommended.

To help Pandoc recognize figures, each figure should keep only one \verb|\caption| and avoid complex patterns such as bilingual caption counter rollback. Note that Word does not support PDF images, so the script first exports PDF images to PNG during processing.

\subsection{表格示例 Table Example}
\label{subsec:table}

表\ref{tab:sample}给出一个三线表。表格中可以使用\verb|booktabs|的\verb|\toprule|、\verb|\midrule|和\verb|\bottomrule|，这类结构通常比复杂合并单元格更利于转换。 Table \ref{tab:sample} gives a three-line table. Tables can use the \verb|\toprule|, \verb|\midrule|, and \verb|\bottomrule| commands from \verb|booktabs|; this structure is usually easier to convert than complex merged cells.

\section{结论 Conclusion}
\label{sec:conclusion}

本文示例保留了论文手稿的核心语义元素，同时删除了期刊模板中的页眉页脚、双栏、字体、复杂标题页和专用参考文献宏。这样的写法更适合作为Pandoc转Word审阅稿的源文件。 This example keeps the core semantic elements of a manuscript while removing journal-template elements such as headers and footers, two-column layout, special fonts, complex title pages, and dedicated reference macros. This writing style is more suitable as the source file for a Pandoc-generated Word review draft.
